1)Number-Represent the Numeric Values
2)String-Represent sequence of letters

4)Undefined-Represent the undefined values
5)Null-Represent no values
6)Object-Represents instance through which we can access members
7)Bigint-Represent larger whole numbers  
8)symbol- To add unique property keys to an object that won't collide with keys any other code might add to the object
